# InternIQ: Internship Job Market Analysis Dashboard

## Project Overview

This project analyzes internship opportunities to uncover hiring trends, salary patterns, work-mode preferences, and geographic demand. The project follows a complete data analytics workflow including data collection, cleaning, SQL analysis, and dashboard development.

## Tech Stack

* Python
* Pandas
* PostgreSQL
* SQL
* Power BI

## Project Workflow

### Data Collection

* Collected internship listing data from online internship platforms.
* Stored data in CSV format for processing.

### Data Cleaning & Feature Engineering

* Removed duplicate records.
* Handled missing values.
* Standardized column names.
* Created:

  * Min_Stipend
  * Max_Stipend
  * Avg_Stipend
  * Work_Mode
  * City
  * Posting_Days_Ago

### SQL Analysis

Performed analysis to identify:

* Highest paying internship categories
* Top hiring cities
* Top hiring companies
* Salary distribution
* Work mode trends

### Power BI Dashboard

#### KPIs

* Total Openings: 635
* Average Stipend: ₹12.3K
* Companies Hiring: 458
* Remote Roles: 273

#### Dashboard Visuals

* Highest Paying Categories
* Top Hiring Cities
* Top Hiring Companies
* Salary Distribution
* Work Mode Distribution

## Key Insights

* DevOps Internship is the highest paying category.
* Bangalore is the top hiring city.
* Remote opportunities account for approximately 43% of listings.
* ₹10K–15K is the most common stipend range.
* Technical roles generally offer higher compensation.
